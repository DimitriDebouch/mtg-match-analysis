# mtg-match-analysis
🎯 Sobre o Projeto

Este projeto consiste numa análise de dados detalhada sobre as partidas de Magic: The Gathering (formato Commander) realizadas por um grupo de amigos. O objetivo foi transformar registos informais de jogos num ecossistema de dados estruturado para identificar padrões de vitória, eficiência de baralhos e a evolução do metagame entre Abril e Novembro.

O projeto demonstra competências em Engenharia de Dados (estruturação de bases), Análise Estatística e Business Intelligence (visualização de KPIs).
📊 Indicadores e Insights Extraídos

A análise foi feita com base em 169 jogos e 219 baralhos únicos, revelando os seguintes pontos:

    Taxa de Vitória (Win Rate): Identificação dos decks mais consistentes, como o Gale, Waterdeep Prodigy, que alcançou 58,62% de aproveitamento.

    Vantagem do Primeiro Turno: Os dados provaram que o jogador que inicia a partida tem uma probabilidade de vitória de 58,27%, vantagem esta associada a modalidade de jogo.

    Distribuição de Cores: Análise do equilíbrio do "meta", com o Vermelho e o Azul (21% cada) sendo as cores mais presentes em comandantes vitoriosos.

    Velocidade do Jogo: A vitória mais rápida foi registada em apenas 7 minutos, com o menor número de turnos para o fecho do jogo sendo 4 turnos.

    Subtipos Populares: Domínio de criaturas do tipo Human Wizard (73 jogos) e Human Warrior.

🛠️ Tecnologias Utilizadas

    Excel/Google Sheets: Utilizado como base de dados inicial para o registo granular das partidas.

    Power BI: Modelagem de dados, criação de medidas em DAX e desenvolvimento de dashboards interativos.

    Estatística Descritiva: Cálculo de médias móveis de participação, correlação entre tempo de jogo e número de jogadores.

📉 Visualizações (Dashboards)

O relatório foi dividido em visões estratégicas:

    Visão Geral: Volume de jogos por mês e média de jogadores por partida (estabilizada em ~4.3).

    Visão de Desempenho: Avaliação do desenvolvimento dos jogos e jogadores ao longo do tempo.

    Ranking de Comandantes: Comparativo de performance entre diferentes estratégias (Tribal, Big Mana, Spellslinger).

    Análise de Jogador (Deep Dive): Filtros personalizados para analisar a performance individual.

    Tempo e Turnos: Estudo da duração das partidas para entender o ritmo do grupo.

    Modalidade de jogo: Filtro usado para segmentar e analisar as diferentes modalidades de jogo e como elas se relacionam com a vitória, duração e desempenho dos jogadores.
